# Hemant Singh – Developer Portfolio

A modern futuristic portfolio website showcasing my work in AI/ML, DevOps, Cybersecurity, Data Science, and Software Development.

## Features

- Animated tech particle background
- Typing terminal effect
- Interactive UI animations
- Skill progress visualization
- Modern cyberpunk UI design
- Fully responsive layout

## Tech Stack

Frontend
- HTML5
- CSS3
- JavaScript

Tools
- Vite
- Node.js
- NPM

Libraries
- Lucide Icons
- Google Fonts

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/hemant-portfolio.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
